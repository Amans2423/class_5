1️⃣ How Does a Computer Store Information?

🟢 What is "Information"?

Anything you type, draw, save, or create on a computer is called information.

💾 What is Memory?

Just like your brain remembers things, a computer uses memory to remember your work.

📌 Two Types of Memory:

  • 🧠 RAM (Primary Memory): Temporary memory — used while working.
  • 📦 Hard Disk (Secondary Memory): Permanent memory — saves things for future use.

✏️ Example: Writing a Story

  1. You open Word and start typing — goes to RAM.
  2. Power off without saving? ❌ Lost!
  3. Click "Save" — goes to Hard Disk and stays safe.

🔍 Memory Comparison Table

Memory Type Speed Power Off Safe?
RAM (Primary) Very Fast ❌ No
Hard Disk (Secondary) Slower ✅ Yes

🧠 Real Life Comparison

  • RAM = Study Table – used while working
  • Hard Disk = School Bag – keeps books safe for later
